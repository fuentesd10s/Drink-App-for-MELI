Drink app for MELI is an app showing a list of drinks using a public API giving a cool design.

- Retrofit
- Fragments
- Glide
- Navigation component
- Material Design
- Coroutines
- MVVM
- ViewModel & LiveData
- RecyclerView, ConstraintLayout, LinearLayout, CardView, etc
- ViewBinding
